# Кейс-задача №1: Стилистическое преобразование чисел (Конвертер даты)

## 📌 Описание проекта
Консольное приложение на Python, которое запрашивает у пользователя день, месяц и год рождения, выполняет ряд вычислений и отображает дату рождения в стилизованном виде — крупными цифрами, составленными из символов «*» (имитация электронного табло).

Проект разработан в рамках учебной практики (технологическая (проектно-технологическая) практика) и демонстрирует навыки работы с датами, функциями, условной логикой, а также творческий подход к выводу данных.

---

## 🎯 Требования к задаче
- Программа последовательно запрашивает день, месяц и год рождения.
- Реализованы функции:
  - определение дня недели для введённой даты;
  - проверка, был ли год високосным;
  - вычисление текущего возраста пользователя в полных годах.
- Вывод даты рождения в формате «дд мм гггг», где каждая цифра нарисована звёздочками (`*`) в стиле семисегментного индикатора (как на электронном табло).
- Код должен быть размещён в публичном репозитории GitHub.

---

## 🛠 Используемые технологии
- **Python 3** – основной язык программирования.
- **Модуль `datetime`** – для работы с датами и определения дня недели.
- **Стандартная библиотека** – никаких сторонних зависимостей.

---

## 📋 Функциональность
1. Ввод даты рождения (день, месяц, год).
2. Определение дня недели (на русском языке, в правильном падеже: «Вы родились в четверг»).

4. Расчёт полных лет на текущую дату с корректным склонением слова «год» (например, «21 год», «3 года», «5 лет»).
