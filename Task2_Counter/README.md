# Кейс-задача №2: Интерактивный счётчик

## 📌 Описание проекта
Веб-приложение «Счётчик», разработанное на чистом HTML, CSS и JavaScript. Представляет собой простой счётчик с двумя кнопками («+» и «–») и окном отображения текущего значения. Интерфейс стилизован с использованием градиентов, кнопки имеют скругление, а цвет фона результата меняется в зависимости от знака числа. При достижении значений 10 или –10 кнопки блокируются, а внизу появляется предупреждающее сообщение.

Проект создан в рамках учебной практики (технологическая (проектно-технологическая) практика) и демонстрирует навыки вёрстки, обработки событий и динамического изменения DOM.

---

## 🎯 Требования к задаче
- Интерфейс содержит две кнопки («плюс» и «минус») и окно для отображения результата.
- Фон страницы – линейный градиент (произвольные цвета).
- Элементы центрированы относительно окна браузера.
- Кнопки имеют радиус скругления 5 пикселей и градиентный фон.
- Логика на ванильном JS:
  - при нажатии «плюс» значение увеличивается на 1, «минус» – уменьшается на 1;
  - цвет окна результата: жёлтый для положительных, зелёный для отрицательных, красный для нуля;
  - при достижении 10 кнопка «плюс» становится неактивной, при –10 – неактивной становится кнопка «минус»;
  - при значениях ±10 под кнопками появляется надпись «вы достигли экстремального значения».

---

## 🛠 Используемые технологии
- **HTML5** – структура страницы.
- **CSS3** – стилизация (градиенты, flexbox, тени, адаптив).
- **JavaScript (vanilla)** – обработка событий, манипуляции с классами и атрибутами.

---

## 📋 Функциональность
1. **Счётчик** – текущее значение отображается в центре страницы.
2. **Кнопки управления**:
   - «+» – увеличение значения на 1.
   - «–» – уменьшение значения на 1.
3. **Изменение цвета фона**:
   - положительное число → жёлтый фон;
   - отрицательное число → зелёный фон;
   - ноль → красный фон.
4. **Блокировка кнопок**:
   - при значении 10 кнопка «+» становится неактивной (disabled);
   - при значении –10 кнопка «–» становится неактивной;
   - в остальных случаях обе кнопки активны.
5. **Сообщение об экстремальном значении**:
   - при значениях 10 или –10 под кнопками появляется текст: «⚠️ Вы достигли экстремального значения!».
