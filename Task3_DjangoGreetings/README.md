# Персонализированное приветствие (Django-приложение)

Веб-приложение на Django, которое позволяет пользователю ввести своё имя, сохраняет его в базе данных и выводит персонализированное приветствие на главной странице. Реализована обработка ошибок (пустое поле), защита от CSRF, стилизация с помощью CSS.

## Функциональность
- Форма ввода имени с кнопкой «Отправить».
- Сохранение имени в базу данных (модель `UserName` с полем `name`).
- Отображение приветствия после успешной отправки: «Привет, {имя}!».
- Валидация: если поле пустое, форма возвращает ошибку.
- CSRF-защита (встроенная в Django).
- Стилизованный интерфейс (градиенты, тени, адаптивность).

## Технологии
- Python 3.x
- Django 4.x / 5.x
- SQLite (база данных по умолчанию)
- HTML5, CSS3

## Установка и запуск

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/your-username/Task3_DjangoGreetings.git
   cd Task3_DjangoGreetings
   ```

2. **Создайте и активируйте виртуальное окружение**
   - Windows:
     ```bash
     python -m venv venv
     venv\Scripts\activate
     ```
   - macOS/Linux:
     ```bash
     python3 -m venv venv
     source venv/bin/activate
     ```

3. **Установите зависимости**
   ```bash
   pip install -r requirements.txt
   ```
   Если файла `requirements.txt` нет, установите Django вручную:
   ```bash
   pip install django
   ```

4. **Примените миграции**
   ```bash
   python manage.py migrate
   ```

5. **Запустите сервер разработки**
   ```bash
   python manage.py runserver
   ```

6. **Откройте в браузере**
   Перейдите по адресу [http://127.0.0.1:8000/](http://127.0.0.1:8000/)
