# Персонализированное приветствие (Django-приложение)

Веб-приложение на Django, которое позволяет пользователю ввести своё имя, сохраняет его в базе данных и выводит персонализированное приветствие на главной странице.  
Реализована обработка ошибок (пустое поле), защита от CSRF, стилизация с помощью CSS.

---

## Функциональность

- 📝 Форма ввода имени с кнопкой «Отправить».
- 💾 Сохранение имени в базу данных (модель `UserName` с полем `name`).
- 👋 Отображение приветствия после успешной отправки: **«Привет, {имя}!»**.
- ⚠️ Валидация: если поле пустое, форма возвращает ошибку.
- 🛡️ CSRF-защита (встроенная в Django).
- 🎨 Стилизованный интерфейс (градиенты, тени, адаптивность).

---

## Установка и запуск

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/your-username/Task3_DjangoGreetings.git
   cd Task3_DjangoGreetings
   ```

2. **Создайте и активируйте виртуальное окружение**
   - Windows:
     ```bash
     python -m venv venv
     venv\Scripts\activate
     ```
   - macOS/Linux:
     ```bash
     python3 -m venv venv
     source venv/bin/activate
     ```

3. **Установите зависимости**
   ```bash
   pip install -r requirements.txt
   ```
   Если файла `requirements.txt` нет, установите Django:
   ```bash
   pip install django
   ```

4. **Примените миграции**
   ```bash
   python manage.py migrate
   ```

5. **Запустите сервер разработки**
   ```bash
   python manage.py runserver
   ```

6. **Откройте в браузере**
   Перейдите по адресу [http://127.0.0.1:8000/](http://127.0.0.1:8000/)

## Детали реализации

- **Модель** `UserName` содержит одно текстовое поле `name`.
- **Форма** `NameForm` создана на основе модели с дополнительной валидацией (проверка на пустое значение).
- **Представление** `index` обрабатывает GET и POST запросы. При успешном POST‑запросе имя сохраняется в БД, и происходит перенаправление с параметром `?name=...` для отображения приветствия.
- **Шаблон** использует тег `{% csrf_token %}` для защиты от CSRF-атак.
- **Стили** подключены через статический файл `style.css` и обеспечивают современный вид (градиентный фон, тени, адаптивность).
