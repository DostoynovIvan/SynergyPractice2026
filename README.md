## Описание
Репозиторий содержит решения пяти кейс-задач, выполненных в рамках учебной практики.

## Структура и задачи

| Папка					   | Краткое описание																	  | Технологии				  |
|--------------------------|--------------------------------------------------------------------------------------|---------------------------|
| `Task1_DateConverter`    | Конвертер даты рождения: день недели, високосность, возраст, вывод даты звёздочками. | Python                    |
| `Task2_Counter`          | Интерактивный счётчик с изменением цвета и блокировкой кнопок при ±10.               | HTML, CSS, JS             |
| `Task3_DjangoGreetings`  | Веб-приложение на Django: форма ввода имени, сохранение в БД,						  |							  |
|	     				   | персонализированное приветствие, обработка ошибок, CSRF-защита.					  | Django, SQLite, HTML, CSS |
| `Task4_Calculator`	   | Калькулятор с четырьмя операциями и проверкой ввода на числа.					      | HTML, CSS, JS             |
| `Task5_AnalyticalReview` | Аналитический обзор одной из задач (по выбору) по критериям качества.				  | —------------------------ |

## Запуск проектов
- Для задач 1, 2, 4 достаточно открыть соответствующий HTML-файл в браузере или запустить Python-скрипт.
- Для задачи 3 (Django) см. инструкцию в папке `Task3_DjangoGreetings/README.md` (если есть) или выполните стандартные команды Django.
