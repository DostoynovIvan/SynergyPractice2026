# Кейс-задача №4: Интерактивный калькулятор

## 📌 Описание проекта
Веб-приложение «Интерактивный калькулятор», разработанное в рамках учебной 
практики (технологическая (проектно-технологическая) практика).  
Калькулятор позволяет выполнять четыре базовых арифметических операции над двумя 
числами, введёнными пользователем, с проверкой корректности ввода и обработкой 
ошибок (деление на ноль, нечисловые символы).

---

## 🎯 Требования к задаче
- Интерфейс включает два поля ввода (`input`), четыре кнопки с названиями
операций («Сумма», «Разность», «Произведение», «Деление») и блок для вывода результата.
- Стилизация интерфейса на усмотрение исполнителя (в данном проекте использованы
градиенты, тени, адаптивная вёрстка).
- Реализованы четыре функции, соответствующие математическим операциям.
- При нажатии на кнопку считываются значения из полей ввода, производится проверка
на корректность чисел, выполняется операция, результат отображается в блоке.
- Если в поля введены не числа (или поля пустые), в блоке результата выводится 
сообщение об ошибке: *«Ошибка: введите числа»*.
- При попытке деления на ноль выводится сообщение: *«Ошибка: деление на ноль»*.

---

## 🛠 Используемые технологии
- **HTML5** – структура страницы.
- **CSS3** – стилизация (градиентный фон, скруглённые кнопки, тени, центрирование).
- **JavaScript (vanilla)** – логика калькулятора, обработчики событий, валидация ввода.

---

## 📋 Функциональность
- Поля ввода принимают любые символы, но перед вычислением происходит проверка: значения
преобразуются в числа с помощью `parseFloat`, и если результат `NaN` – выбрасывается ошибка.
- Кнопки:
  - **Сумма** – сложение двух чисел.
  - **Разность** – вычитание второго числа из первого.
  - **Произведение** – умножение.
  - **Деление** – деление первого числа на второе (с защитой от деления на ноль).
- Результат отображается в формате: `Результат: <значение>`.
- При ошибках текст в блоке результата заменяется соответствующим сообщением.

## 🚀 Запуск приложения
1. Скачайте или клонируйте репозиторий.
2. Откройте папку `Task4_Calculator`.
3. Запустите файл `index.html` в любом современном браузере (Chrome, Edge, Firefox, Safari).
4. Введите числа в поля и нажимайте кнопки для получения результата.

**Внимание:** для корректного отображения стилей необходимо, чтобы файлы `style.css` и 
`script.js` находились в той же папке, что и `index.html`.
